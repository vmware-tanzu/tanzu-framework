# pinniped-config-controller-manager

This directory is home to the `pinniped-config-controller-manager`.

To run this on a cluster:

```sh
./hack/run.sh
```

To run the tests:

```sh
./hack/test.sh
```

To run static checks (e.g., `go fmt`, `go vet`, `golangci-lint`) and the tests:

```sh
./hack/check.sh
```
